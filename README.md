# **CRUD - TypeORM**
## Sistema de listagem de posts utilizando typescript.

## **1 - Frontend**
### Tecnologias utilizadas
* HTML.
* CSS.
* Javascript.
* Fetch API.
### Features:
* Darkmode salvo no localstorage.
* Design responsivo.
* Capacidade de listar, criar, buscar, deletar e dar like em posts.
